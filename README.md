# Ženska Pijaca

Dobrodošli u Ženska Pijaca projekat.

## O projektu

Online pijaca ručno izrađenih proizvoda od srpskih preduzetnica. Organska hrana, prirodna kozmetika, med, džemovi i unikatne rukotvorine.

## Kako pokrenuti projekat

Potrebno je da imate instaliran Node.js & npm.

Pratite sledeće korake:

```sh
# Korak 1: Instalirajte zavisnosti
npm install

# Korak 2: Pokrenite razvojni server
npm run dev
```

## Tehnologije

Ovaj projekat koristi:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
